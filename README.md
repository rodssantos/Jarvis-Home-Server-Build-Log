# Jarvis Home Server Build Log

This repository documents the step-by-step build of a **self-hosted AI Home Server ("Jarvis")**.

The goal is to create a private system capable of:

- AI assistance (local LLMs)
- photo intelligence (Immich)
- document understanding (RAG)
- media streaming
- personal cloud storage
- automation and voice interaction

# 🚀 Current System Status

Deployed components:

- Ubuntu Server
- Docker + Docker Compose
- Portainer
- Traefik (reverse proxy)
- Immich (AI photo server)
- PostgreSQL + Redis
- Machine Learning container (CPU-based)
